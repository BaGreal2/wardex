# WARDEX - A small device that protects your home from robbery.

### Features:
- The device should be attached to the door panel with sticky tape and reacts to the door movement; 
- User can manipulate the behaviour of device using app;
- Littles speakers for sound alert;
- The device with sizze of human finger;
- NFC module for autorization;

### What's in the box?

- WARDEX device;
- Two coin cell batteries;
- A set of sticky tapes;
- Instructions;

### How it works?

Device is statically attached to the door/window panel any way our client wants but directed with detector to the moving part of door or window. The device will detect any opening movement and send signal to the servers and then to the user end device. The detection performs by the laser inside the device body. The client will be notified in app that someone tries to enter the home. User can switch on/off notifications if for example he needs to open the door. Also phusical device by default will make sound additionally to the app notification. 

### App Features:

- Turn on/off alert;
- Turn on/off speakers;
- See the battery percentage;
- Call the police;
